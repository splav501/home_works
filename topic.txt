                                                Each VS Map in Ruby

Для начала давайте разберемся с понятием Блок.
Блок — это произвольный код, который можно передать любому методу в качестве неявного последнего аргумента. Блок
разграничивается фигурными скобками {} или ключевыми словами do end.

                                                     Each

Each - это метод который совершает перебор по каждому элементу Array или Range не изменяя его.

Для премера объявим массив cities_arr внутри имеющего название городов и приименим метод Each для прохождения по каждому
элементу массива и вызовим метод puts для вывода в консоль.

cities_arr = ['washington', 'roma', 'paris', 'london', 'berlin']
cities_arr.each {|city| puts city}
#=> ["washington", "roma", "paris", "london", "berlin"]


                                                   Map and Map!

Теперь давайте расмотрим метод Map также как и Each делает перебор по каждому элементу Array или Range и возвращает новый
массив. Метод map! изменяет исходный массив.

Вы наверное обратили внимание что в созданном нами cities_arr все элементы начинаются с маленькой буквы. Так давайте же
это исправим с помощью метода map и capitalize. Результат присвоим новому массиву cities_capitalized.

cities_capitalized = cities_arr.map{|city|  city.capitalize}
puts cities_capitalized
#=> ["Washington", "Roma", "Paris", "London", "Berlin"]

Если же мы хотим хотим изменить исходный массив cities_arr, то используем метод map!.

cities_arr.map!{|city| city.capitalize}
puts cities_arr
#=> ["Washington", "Roma", "Paris", "London", "Berlin"]


Теперь нам известно, что матоды Each и Map оба проходятся по каждому элементу массива с которым мы можем оперировать,
разница лиш заключается в том, что Map в результате возвращает массив, а Each ничего не возвращает.
